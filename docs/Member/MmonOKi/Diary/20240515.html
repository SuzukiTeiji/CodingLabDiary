問題1
テンプレートリテラルってどんなもの？
回答
テンプレートリテラルは、バッククォート（`）で囲まれた文字列内で変数や式を埋め込む方法です。例えば、console.log(`${name}は${age}歳です。`);こんな感じで書きます。

問題2
以下のプログラムを読んで、consoleに表示される文字列を予想してみよう。
let number = 10;
if (number > 5) {
console.log("数字は5より大きいです。");
} else {
console.log("数字は5以下です。");
}
回答
数字は5より大きいです。
