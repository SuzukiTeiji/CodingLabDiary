問題1
配列とはどんなもの？
回答
配列は、順序付けられたデータのリストです。各データは「要素」と呼ばれ、0から始まるインデックスでアクセスできます。配列は[ ]を使って作成します。例えば、let fruits = ["りんご", "バナナ", "オレンジ"]; のように定義します。

問題2
オブジェクトとはどんなもの？
回答
オブジェクトは、名前（キー）と値のペアを持つデータの集合です。複数の関連するデータを1つの変数にまとめるのに便利です。オブジェクトは{ }を使って作成します。例えば、let person = { name: "太郎", age: 30, job: "エンジニア" }; のように定義します。
